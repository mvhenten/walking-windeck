# matshikes

Mobile-friendly hiking map for the hills around Dreisel / Windeck (and anywhere else).

- OpenTopoMap base with contours + hill shading
- Waymarked hiking trails overlay
- Tap-to-draw paths that snap to real footpaths via [BRouter](https://brouter.de)
  (`hiking-mountain` profile)
- Live distance / ascent / descent stats
- Save / load routes (currently `localStorage`; Google Drive sync coming)
- GPS locate

Open the deployed page on a phone and "Add to Home Screen" for an app-like feel.
